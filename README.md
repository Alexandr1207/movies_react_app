# Movie Search App (React + Material UI)

Интерактивное приложение для поиска и фильтрации фильмов с использованием **TMDB API**. Проект позволяет пользователям просматривать список популярных фильмов, фильтровать их по жанрам, годам выпуска, сортировать по рейтингу и популярности, а также добавлять в "Избранное".

## Ссылка на проект (GitHub Pages)
https://alexandr1207.github.io/movies_react_app/

## Основные функции

* **Динамическая фильтрация:** Поиск фильмов по названию, жанрам (через Autocomplete) и диапазону лет (Slider).
* **Сортировка:** Возможность упорядочить список по популярности или среднему рейтингу.
* **Пагинация:** Удобная навигация по страницам выдачи API.
* **Избранное (Optimistic UI):** Возможность добавлять/удалять фильмы из списка избранных с мгновенным обновлением интерфейса.
* **Детальная страница фильма:** Просмотр подробной информации, актерского состава и бюджета.

## 🛠 Технологический стек

* **Frontend:** React (Vite)
* **UI Library:** Material UI (MUI)
* **Icons:** MUI Icons
* **Routing:** React Router DOM (HashRouter)
* **API:** The Movie Database (TMDB)
* **State Management:** Context API + React Hooks (useState, useEffect)

## Установка и запуск

Чтобы запустить проект локально, выполните следующие шаги:

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Alexandr1207/movies_react_app/
2. Установите зависимости:
   ```bash
   npm install
3. Создайте файл .env в корневой папке и добавьте ваши ключи API (проект использует префикс VITE_ для доступа к переменным):
   ```bash
   VITE_TMDB_TOKEN=ваш_access_token_здесь
   VITE_TMDB_ACCOUNT_ID=ваш_account_id_здесь
4. Запустите проект в режиме разработки:
   ```bash
   npm run dev

## Деплой

  npm run deploy

## Особенности реализации:
* **Optimistic UI:** В приложении реализована логика мгновенного переключения иконки "Избранное" до получения ответа от сервера. В случае ошибки запроса состояние автоматически откатывается назад, что обеспечивает лучший UX.
* **Clean Code:** Логика API вынесена в отдельные асинхронные функции, используются современные методы обработки массивов и объектов.
